# 🧩 Puzzle Adventure

An interactive web-based puzzle adventure game that combines storytelling, problem-solving, and exploration. Players progress through multiple puzzle levels while following a narrative-driven adventure.

## 🌟 Features

### 📖 Story-Based Gameplay

* Engaging storyline that guides players through the adventure.
* Progressive challenges tied to the narrative.

### 🧩 Multiple Puzzle Levels

* Six puzzle stages with unique challenges.
* Players must solve each puzzle to continue their journey.

### 🎨 Interactive User Interface

* Responsive design using HTML, CSS, and JavaScript.
* Clean and intuitive gameplay experience.

### ⚡ Dynamic Game Logic

* Puzzle validation and progression handled using JavaScript.
* Interactive feedback for user actions.

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 🚀 How to Run

1. Download or clone the repository.

```bash id="3n0k1h"
git clone <repository-url>
```

2. Open `story.html` in a web browser.

3. Follow the story and solve the puzzles to complete the adventure.
